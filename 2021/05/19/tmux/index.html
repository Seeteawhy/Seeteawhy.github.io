tmux

发表于2021-05-19|更新于2021-05-23
|阅读量:

之前用element-UI写了个后台管理系统，用tmux直接跑在服务器上，感觉挺好用，分享一下。

tmux是什么

tmux的作用就是终端窗口关闭时，会话并不终止，而是继续运行，等到以后需要的时候，再让会话”绑定”其他窗口。
Tmux 就是会话与窗口的”解绑”工具，将它们彻底分离。
让我感觉最强大的功能是：它允许每个会话有多个连接窗口，可以多人实时共享会话。

tmux怎么用

由于我租的vps是ubuntu系统，所以以它为例。

安装

$ sudo apt-get install tmux

启动与退出

$ tmux
这时，终端窗口下面会出现绿条

$ exit
或者ctrl + d 退出

新建会话

$ tmux new -s
默认启动tmux窗口，它的session-name是0，以后为1，2…这样不方便管理，所以建议取名

分离会话

在 Tmux 窗口中，按下Ctrl+b d或者输入tmux detach命令，就会将当前会话与窗口分离。会话和里面的进程仍然在后台运行。
tmux ls命令可以查看当前所有的 Tmux 会话。

接入会话

$ tmux attach -t

杀死会话

$ tmux kill-session -t

切换会话

$ tmux switch -t

以上就是tmux最最基本的操作啦，约等于0的学习成本，功能强大，非常推荐。
